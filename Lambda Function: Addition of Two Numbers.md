To write a Python program that defines a **lambda function** which takes two arguments `a` and `b`, and returns their sum.

## ðŸ§  Algorithm

## ðŸ§¾ Program
```

num1 = int(input("Enter first number: "))

print(f"The sum of {num1} and {num2} is: {result}")
```
